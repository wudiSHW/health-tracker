# 🏥 健康记录助手

一个专为病人设计的健康管理PWA应用，可以记录血压、吃药和运动情况，并提供定时提醒功能。

## ✨ 功能特点

### 📊 健康记录
- **血压记录**: 支持早中晚三次血压测量记录（收缩压/舒张压）
- **吃药记录**: 记录早中晚三次服药情况
- **运动记录**: 记录运动类型和时长

### ⏰ 智能提醒
- 自定义血压测量提醒时间
- 自定义吃药提醒时间
- 自定义运动提醒时间
- 本地通知推送（需要授权）

### 📈 数据统计
- 实时统计今日记录次数
- 运动总时长统计
- 历史记录查看（最近7天）
- 数据导出为Excel/CSV格式

### 📱 PWA特性
- 可安装到手机桌面（安卓/iOS）
- 离线使用支持
- 本地数据存储
- 响应式设计，适配各种屏幕

## 🚀 安装方法

### 方法一：直接安装（推荐）

1. 在安卓手机上用Chrome浏览器打开应用网址
2. 点击底部弹出的"安装到桌面"提示
3. 或在菜单中选择"添加到主屏幕"
4. 安装完成后，像普通应用一样使用

### 方法二：手动安装

1. 下载本项目所有文件
2. 部署到任意Web服务器（或本地服务器）
3. 在安卓Chrome中访问应用地址
4. 点击菜单 → 添加到主屏幕

## 📋 使用说明

### 记录血压
1. 在"记录"页面找到血压记录卡片
2. 选择对应的时间段（早晨/中午/晚上）
3. 输入收缩压（高压）和舒张压（低压）
4. 点击"记录"按钮保存

### 记录吃药
1. 在"记录"页面找到吃药记录卡片
2. 勾选对应时间段的服药记录
3. 自动保存当前时间

### 记录运动
1. 在"记录"页面找到运动记录卡片
2. 选择运动类型
3. 输入运动时长（分钟）
4. 点击"记录运动"按钮

### 设置提醒
1. 切换到"提醒"页面
2. 选择需要提醒的项目
3. 设置提醒时间
4. 点击"设置提醒"按钮

### 导出数据
1. 切换到"统计"页面
2. 点击"导出Excel"按钮
3. 数据将以CSV格式下载到手机

## 🔧 技术栈

- **前端**: HTML5, CSS3, JavaScript (ES6+)
- **PWA**: Service Worker, Web App Manifest
- **存储**: LocalStorage
- **通知**: Web Notifications API

## 📱 系统要求

- **安卓**: Android 5.0+，Chrome 60+
- **iOS**: iOS 11.3+，Safari
- **桌面**: 支持PWA的现代浏览器

## 🔒 隐私说明

- 所有数据存储在本地，不会上传到服务器
- 通知权限仅用于健康提醒
- 无需注册账号即可使用
